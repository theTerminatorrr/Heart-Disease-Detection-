# Heart-Disease-Detection-
Cardiovascular diseases (CVDs) remain the leading cause of global 
morbidity and mortality. Early and accurate prediction of heart 
anomalies plays a vital role in clinical preventive care ecosystems.
The objective of this laboratory project is to construct a robust 
machine learning diagnostic framework. By analyzing critical 
physiological parameters, this project builds and validates two 
distinct classification pipelines: a generalized linear framework 
(Logistic Regression) and a distance-based instance framework 
(K-Nearest Neighbors). This comparative study evaluates 
performance across multiple dimensions—such as classification 
boundaries, stability, and predictive sensitivity—to establish an 
optimized tool for automated clinical screening support.
